В рамках тестового задания было разработано клиентское приложение для работы с сервисом для хранения телефонных контактов.
Для работы с сервисом необходимо авторизоваться. Для этого нужно заполнить поля Логин и Пароль и нажать кнопку Войти.
Для смены пользователя нужно нажаь кнопку Выйти, ввести логин и пароль и нажать кнопку Войти.
После этого можно приступать к работе с сервисом. Запросы неавторизованных пользователей будут отклюнены сервисом.
Для добавления контакта необходимо заполнить поля в правой части окна клиентского приложения и нажать кнопку Добавить. Заполнение поля ID в этом случае не нужно.
Для поиска контакта по имени необходимо ввести имя в поле для поиска в лево части окна клиентского приложения и нажать кнопку Найти. Найденная запись будет отображена в центре окна клиентского приложения.
Для изменения контакта необходимо заполнить все поля в правой части окна клиентского приложения и нажать кнопку Изменить.
Для удаления контакта необходимо заполнить все поля в правой части окна клиентского приложения и нажать кнопку Удалить.
